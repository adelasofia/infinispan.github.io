Quarkus

Quarkus is a Kubernetes Native Java framework tailored for GraalVM and HotSpot, crafted from best-of-breed Java libraries and standards. The goal of Quarkus is to make Java a leading platform in Kubernetes and serverless environments while offering developers a unified reactive and imperative programming model to optimally address a wider range of distributed application architectures. Quarkus comes with an extension that enables connecting to an Infinispan server.

WildFly

WildFly is a JavaEE application server which uses Infinispan as the base for its clustering needs: web sessions, stateful session beans, etc

Hibernate ORM

Hibernate ORM has a second-level cache provider built on top of Infinispan.

Hibernate OGM

Hibernate OGM can perform CRUD operations on Infinispan, allowing you to replace the traditional RDBMS with Infinispan.

Hibernate Search

THibernate Search can store its Lucene index in Infinispan to provide real time replication of the index.

Vert.x

Vert.x has an Infinispan Cluster Manager, able to delegate its clustering requirements to Infinispan.

Teiid

Teiid is a data virtualization system that allows applications to use data from multiple, heterogenous data stores. In particular, it supports using Infinispan as a data-source.

ModeShape

ModeShape is a distributed, hierarchical, transactional, and consistent data store which adheres to the JCR standard. It comes with an Infinispan connector which allows Infinispan to be used as a repository.

Apache Camel

Camel is a data integration framework which allows you to define routing and mediation rules in a variety of domain-specific languages. It comes with an Infinispan component, so that you can store/retrieve/query data from both embedded and remote caches.

Apache Marmotta

Marmotta is an open implementation of a Linked Data Platform. It uses Infinispan to provide caching for Linked Data resources.

apiman

Apiman is an API management platform which uses Infinispan as a caching layer to increase performance.

CapeDwarf

CapeDwarf is an open-source alternative to Google App Engine, allowing you to run applications written for the Google App Engine on the WildFly Application Server without modification. CapeDwarf uses Infinispan to implement the GAE Datastore API.

Immutant

Immutant is an integrated suite of Clojure libraries. It exposes a caching service built using Infinispan.
