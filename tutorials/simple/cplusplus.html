Access a remote cache on an Infinispan Server using the HotRod protocol in C++

This tutorial requires installing the appropriate HotRod C++ libraries for your platform, a working C++ compiler and CMake
This tutorial requires a locally-running Infinispan Server started with the standalone.sh (Linux, OS X, Unix) or standalone.bat (Windows) script.
